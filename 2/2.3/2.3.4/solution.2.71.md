## 当 n == 5 时

     31

## 当 n == 10 时

```text
          1023
          / \
         511 512
         /  \
        255 256
        /  \
       127 128
       / \
      63 64
      / \
     31 32
     / \
    15 16
    /\
    7 8
   /\
   3 4
  /\
 1  2
```

## 由以上可以观察出

最频繁的数，只需要 1 位二进制位

最不频繁的数，需要 n-1 位二进制位
